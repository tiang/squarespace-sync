Intro to Creative Coding

Transform passive screen time into active creation. This course introduces students aged 10-14 to the fundamental concepts of computer science through the lens of visual art and animation.

Syllabus

Module 1: The Canvas of Code

Introduction to the development environment and coordinate systems. Creating your first digital portrait using shapes and colors.

  • Coordinate Planes
  • Primitive Shapes
  • Color Theory in RGB
  • The Draw Loop

Module 2: Interaction & Variables

Making static art come to life. Learning how to store information and respond to user inputs like mouse clicks and keyboard presses.

Module 3: Logic & Loops

Mastering conditional statements to create interactive stories and simple physics systems.

Module 4: Final Showcase Project

Students design and code a unique interactive experience to present to peers and parents.
